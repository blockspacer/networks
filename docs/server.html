HOW TO CONNECT:

you can build client (located here) using go build

to start it you should place .chat.yml config file in your home directory. config content:

config_dir: /home/sazikov-a/.chat.cfg
server_poll_backoff: 2
server_host: 127.0.0.1
server_port: 37516

in that file you should specify server host and port, directory for storing some data and server polling time in seconds
